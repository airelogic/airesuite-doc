AireGlu Release Notes ​

Version: 1.115.0
Release Date: 10 March 2026

Summary of Changes ​

✨ New Features ​

  • Endpoint Last Saved Timestamp and Sort Options (GLU-634)
  • Endpoint Dependency Graph — Beta (GLU-2832)

⚡ Improvements ​

  • Wrapped Filter Paths in Endpoint Editor (GLU-2924)

🐞 Bug Fixes ​

  • Case-Insensitive Alphabetical Sorting for Endpoints (GLU-1965)

Detailed Changes ​

✨ New Features ​

Endpoint Last Saved Timestamp and Sort Options (GLU-634) ​

Description

The endpoints list now displays when each endpoint was last saved, giving users at-a-glance visibility of recent changes. A new sort dropdown allows endpoints to be ordered by most recently edited or alphabetically.

What this means for you

You can now instantly see when any endpoint was last updated, and quickly sort your endpoints list to bring the most recently changed ones to the top — making it much easier to track and manage ongoing work.

Endpoint Dependency Graph — Beta (GLU-2832) ​

Description

A new dependency graph view is available in beta, allowing users to visualise the relationships between interconnected endpoints. Business transactions often span multiple endpoints, and this graph makes it easier to understand how they depend on one another.

What this means for you

You can now explore a visual map of how your endpoints connect and rely on each other, making it simpler to understand complex workflows and spot the impact of any changes before you make them. As this feature is in beta, feedback is welcome.

⚡ Improvements ​

Wrapped Filter Paths in Endpoint Editor (GLU-2924) ​

Description

Long filter path values in the endpoint editor now wrap across multiple lines rather than overflowing or being truncated, improving readability for complex filter expressions.

What this means for you

Long filter paths are now fully visible in the editor without any need to scroll horizontally, making it easier to review and edit complex endpoint configurations.

🐞 Bug Fixes ​

Case-Insensitive Alphabetical Sorting for Endpoints (GLU-1965) ​

Description

Alphabetical sorting in the endpoints list previously treated uppercase and lowercase letters as separate groups, causing endpoints with upper-case names to be sorted separately from those beginning with lower-case letters. Sorting now correctly treats all letters as case-insensitive.
