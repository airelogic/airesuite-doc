    User Mangagment

    Manage users who have access to your tenant

    Feb 8, 2021

    This page explains how to manage users in AireFrame (AirePolicy). For example you may want to invite a new user to your tenant, or remvoe an existing user.

    What you can find on this page:

    • Required Permissions
    • Add Tenant User
    • Remove Tenant User

    Required Permissions

    • In order to add a user you will need the following (AirePolicy) permissions:
      • Add Tenant User
    • In order to remove a user you will need the following (AirePolicy) permissions:
      • List Tenant Users
      • Remove Tenant User

    Add Tenant User

    1. Go to Aire Policy
    2. Click Authorize, select all the scopes and click Authorize
    3. Login and if applicable, select the tenant that you wish to manage
    4. Go to the add user endpoint (POST /api/v1.0/tenants/{tenantKey}/users)
    5. Enter the tenant key of the tenant you wish to manage (this must match the tenant you selected earlier) and in the body enter the email of the user you wish to add
    6. Click Execute

    If this endpoint responds with a message saying you must invite this user then:

    1. Go to the invite user endpoint (POST /api/v1.0/tenants/{tenantKey}/users/invite)
    2. Enter the tenant key of the tenant you wish to manage (this must match the tenant you selected earlier) and in the body enter the email of the user you wish to add
    3. Click Execute
    4. Once the user signs up, they will be added to your tenant

    Remove Tenant User

    1. Go to Aire Policy
    2. Click Authorize, select all the scopes and click Authorize
    3. Login and if applicable, select the tenant that you wish to manage
    4. Go to the remove user endpoint (DELETE /api/v1.0/tenants/{tenantKey}/users)
    5. Enter the tenant key of the tenant you wish to manage (this must match the tenant you selected earlier) and the user id of the user you wish to delete (see below)
    6. Click Execute

    To get the user id of the user you wish to delete

    1. Go to the gets users endpoint (GET /api/v1.0/tenants/{tenantKey}/users)
    2. Enter the tenant key of the tenant you wish to manage (this must match the tenant you selected earlier)
    3. Click Execute
    4. Find the user you wish to remove and copy the id value
