    Data Source Configuration

    Add and configure the data sources

    Jan 25, 2021

    This page explains how to add data sources in AireFrame. For example you may want to add a FHIR data source so that you can consume subjects or structured data instead of using AireFrame’s internal data store.

    What you can find on this page:

    • Required Permissions
    • Add Data Source
    • Edit Data Source

    Required Permissions

    • In order to add a data source you will need the following permissions:
      • List Data Sources
      • Create Data Source
    • In order to edit a data source you will need the following permissions:
      • List Data Sources
      • View Data Source
      • Edit Data Source

    Add Data Source

    1. Go to the data source configuration screen: Configuration -> Data Sources.
    2. Click Add Data Source, you will see a screen as follows
    3. Select the data source type
    4. Enter a name for the data source
    5. Enter the data source specific configuration values. Required values are indicated with an (*).
    6. Click save

    Edit Data Source

    1. Go to the data source configuration screen: Configuration -> Data Sources.
    2. Click the data source you wish to edit
    3. Enter the values you wish to change. For security reasons, secret values must be re-entered on every edit
    4. Click save
