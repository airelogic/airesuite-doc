    Setting up and modifying groups

    Mar 11, 2021

    This page explains how manage groups in AireFrame. This allows the creation of groupings of structural entities that can then be assigned to users to allow access to specific areas in your organisation.

    What you can find on this page:

    • Required Permissions
    • Groups

    Required Permissions

    • In order to create a group you will need the following permissions:
      • List Groups
      • Create Groups
      • List Structural Entities
    • In order to edit a group you will need the following permissions:
      • List Groups
      • View Groups
      • Edit Groups
      • List Structural Entities

    Groups

    1. Go to the groups configuration screen: Configuration -> Groups.
    2. Click Add Group, you will see a screen as follows
    3. Enter a name for the group
    4. Use the structure tree view to select the structural entites that you want accessible in the group
      • Note that all parents of that entity must be accessible for that entity to be accessible (It is a good idea to assign your subjects at the lowest level possible so that the access control can be finegrained)
    5. Click save

    Groups can be edited by clicking the arrow at the right hand side of the group listing.

    You can assing a group to a user by going to the user management screen: Configuration -> Users, finding the user that you wish to assign to a group and adding the group that you want to assign to them in the groups section of the user screen.
