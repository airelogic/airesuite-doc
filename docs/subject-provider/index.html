    Configure the subject provider to use a custom data source

    Jan 25, 2021

    This page explains how to configure the subject provider in AireFrame. For example you may want to configure that the Subjects are provided by a FHIR data source.

    What you can find on this page:

    • Required Permissions
    • Configure Subject Provider

    Required Permissions

    • In order to configure the subject provider you will need the following permissions:
      • View Subject Configuration
      • Edit Subject Configuration
      • List Data Sources
      • Get Data Source

    Configure Subject Provider

    1. Go to the subject configuration screen: Configuration -> Subject Configuration.
    2. Choose the data source you wish to serve your subjects from the drop down list
    3. Unless you are using the internal data store you will need to provide mappings for each of the custom fields so that the data source can provide the correct data for the subjects - In the case of the FHIR provider these are FHIR Paths.
    4. Click save
