    Structured Data Consumer

    Aug 1, 2020

    In AireFrame you can consume structured data from different data sources and display them in a table or a chart.

    The initial structred data consumers are forms4health form submissions and FHIR R4 observations. In the future we plan to expand the offered data sources to provide further integration points for tenants.

    First you should configure the data source that you want to consume data from - Documentation can be found here.
