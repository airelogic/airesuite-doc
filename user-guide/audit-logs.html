Audit Logging ​

Monitor and log all user activity and system changes on Aire Innovate products, to ensure security, compliance, and operational integrity:

  • AireFrame
  • AireIdentity
  • AireGlu
  • AireFlow
  • AireForms

Result Table ​

After applying filters, the table updates to show matching audit entries. Each entry displays:

  • Timestamp – When the action occurred
  • Product – The Aire Innovate service involved
  • Action – What was performed
  • Outcome – Success or failure
  • Actors – User or system responsible

Clicking the arrow on the left expands the entry to show additional metadata.

Filtering ​

Audit Logs can be filtered using several controls at the top of the page. These filters can be combined to narrow down results and help you locate specific activity quickly.

Product Filter ​

Use the Products dropdown to view audit entries for a single product only. Selecting a product will hide all logs relating to other Aire Innovate services.

Example use cases:

  • View only AireFrame visualisation interactions
  • Review login activity in AireIdentity
  • Narrow logs to a specific system during investigations

Date Range ​

Use the From and To date pickers to filter logs within a specific time window.

  • The From date defaults to the earliest possible time for today.
  • The To date defaults to the latest time on the selected day.

The table updates automatically once both dates are set.

Typical scenarios:

  • Reviewing actions within a security change window
  • Investigating an issue reported at a specific time
  • Exporting activity for a weekly or monthly report

Search ​

The Search bar allows free-text search across audit entries. You can search by:

  • Action name
  • Product
  • Actor

Partial text found within expanded log details. This is useful for tracking down a specific action or event when you know part of its name.

Actor Filter ​

Use the Actor field to filter logs to a single user or system actor.

This narrows the table down to events performed by:

  • A specific user
  • A machine account or system process
