    Data Source Config

    Dec 14, 2020

    Data sources can now be configured on a per-tenant basis. This means one tenant can consume subjects and structured data from an external FHIR server while another tenant keeps all their data in the internal data store. All integrations continue to work whichever data source you decide to consume data form.
